# Student Dashboard Full Stack Project

## 📌 Description
A full stack student dashboard application built using Java (Spring Boot) for backend and HTML/CSS/JS for frontend.

## 🚀 Features
- Add Student
- View Student List
- Update Student Details
- Delete Student
- REST API integration

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Spring Boot (Java)
- Database: MySQL
- Tool: Git & GitHub
